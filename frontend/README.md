# frontend

Cliente CLI da Calculadora de Calorias feito com o template `app` do Leiningen.

## Prerequisitos

Antes de executar, tenha instalado:

- Java JDK.
- Leiningen 2.0.0 ou superior.

## Como Executar

Com a API do backend rodando em `http://localhost:3000`, execute:

```bash
cd frontend
lein run
```

O arquivo principal e o menu ficam em `src/frontend/core.clj`.

## Como Testar

```bash
lein test
```
